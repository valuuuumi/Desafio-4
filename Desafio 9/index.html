Guía para Desplegar API REST en Render

Paso 1: Crear una cuenta en Render

Registrarse en Render mediante su página web y seguir el proceso de creación de una cuenta.

Paso 2: Preparar el proyecto para el despliegue

  • Asegurarse de que el proyecto esté correctamente configurado y funcione localmente.
  • Verificar que todas las dependencias estén listadas en el archivo package.json y que se incluyan en el repositorio.

Paso 3: Configurar el despliegue en Render

  • Acceder al panel de Render y crear un nuevo servicio.
  • Seleccionar las opciones adecuadas para el tipo de servicio que se está desplegando.
  • Configurar las variables de entorno necesarias.

Paso 4: Desplegar el proyecto en Render

  • Conectar el servicio de Render con el repositorio de GitHub o GitLab donde se encuentra el proyecto.
  • Configurar la rama y la configuración de despliegue automático si es necesario.
  • Iniciar el despliegue del proyecto en Render y verificar que se complete correctamente.

Paso 5: Verificar la API REST desplegada

Una vez que el despliegue se haya completado, acceder a la URL proporcionada por Render para verificar que la API REST esté accesible y funcionando correctamente en el entorno de producción.

Bonus Extra

  • Realizar pruebas adicionales en la API REST desplegada.
  • Documentar cualquier problema encontrado durante el proceso de despliegue y cómo se resolvieron.
  • Tomar capturas de pantalla o grabar un video del proceso de despliegue y compartirlo como evidencia adicional.
